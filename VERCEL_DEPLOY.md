# Vercel Deployment for EcoFashion AI

## Quick Deploy to Vercel

Click the button below to deploy this project to Vercel:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/deoxy-genz/EcoFashion-ai)

## Manual Deployment Steps

1. **Go to [Vercel](https://vercel.com/)**
2. **Sign up/Login** with your GitHub account
3. **Import your repository**: `https://github.com/deoxy-genz/EcoFashion-ai`
4. **Configure settings** (all defaults should work)
5. **Deploy** - Your site will be live in minutes!

## Configuration

The project includes:

- `vercel.json` - Deployment configuration
- Static file serving optimized for SPA
- Automatic HTTPS and global CDN

## Live URL

After deployment, your site will be available at:
`https://your-project-name.vercel.app`

## Environment Variables

No environment variables required for basic deployment. The weather API key can be configured through the settings panel in the application.
