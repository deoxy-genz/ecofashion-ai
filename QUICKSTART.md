## 🏃‍♂️ **Immediate Launch Options**

### Option 1: Double-Click Start (Windows)

1. **Double-click** `start-dev-server.bat`
2. Wait for the server to start
3. Your browser will automatically open to `http://localhost:3000`

### Option 2: VS Code Task (Recommended for Development)

1. **Open in VS Code**: File → Open Folder → Select this project
2. **Run Task**: Press `Ctrl+Shift+P` → Type "Tasks: Run Task" → Select "Start EcoFashion AI Development Server"
3. **Access**: Open browser to `http://localhost:3000`

### Option 3: Command Line

```bash
cd "C:\Users\Admin\Desktop\Mini_project"
npm start
```
